Tests are run with Protractor.
For more information visit 
http://angular.github.io/protractor/#/tutorial

To run the tests, do the following:

1. Install node.js
2. Install protactor.js
3. Run 'webdriver-manager update' in command prompt to install the Selenium Server
4. Set the proper URL Address for you local site in conf.js - the global variable browser.erpSiteAddress
5. Run the needed webserver component
6. To run the test do 'test' in 'the WebVella.ERP.Web\wwwroot\tests' folder command prompt
