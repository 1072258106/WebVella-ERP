TIP: Dynamic views are automatically triggered by using a naming convention with the ~ sign. Example: A view with the name area_name~field_name~value123 will be used automatically to display the record details, when the area name is area_name and the record's field field_name has the value of value123. The only drawback is that the values of the field should be only alphanumberic chars.
